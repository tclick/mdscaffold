# Prototyping Notebooks

This directory contains interactive scratchpads used during the development of `mdscaffold`.

* `01_pydantic_validation_tests.ipynb`: Prototyping schema bounds for temperature and replica limits.
* `02_amber_template_formatting.ipynb`: Verifying string substitution logic for `.in` files.
* `03_cpptraj_output_analysis.ipynb`: Parsing synthetic `rmsd.dat` files to prototype core analysis functions.

*Note: Please clear all execution outputs (`Kernel -> Restart & Clear All Outputs`) before committing changes.*
